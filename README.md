# Hasnat Evan Portfolio

Next.js app for Vercel deployment.

## Local Commands

```bash
npm install
npm run dev
npm run lint
npm run build
npm run start
```

## Vercel Deploy

1. Push code to GitHub.
2. Import the repo in Vercel.
3. Set required environment variables in Vercel Project Settings:
- `MONGODB_URI`
- `MONGODB_DB`
- `NODEMAILER_USER`
- `NODEMAILER_PASS`
- `NODEMAILER_FROM` (optional)
- `NODEMAILER_TO` (optional)
- `NODEMAILER_HOST` (optional)
- `NODEMAILER_PORT` (optional)
4. Deploy.
